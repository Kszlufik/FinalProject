PlayPal

A social platform for game tracking, discovery and sharing.

● Live App ⬡ GitHub Flutter Web Firebase RAWG API Steam API

About

PlayPal is a gaming platform where you can discover games, track what you are playing, connect your Steam library, and see what your friends are up to — all in one place. Built as a final year project at SETU using Flutter Web and Firebase.

Everything on the live site is real data. No mocks, no placeholder content.

Features

  • → Game discovery via RAWG API (500k+ titles)
  • → Reviews, ratings and play status tracking
  • → Steam library and achievement import
  • → Friends system with profiles and requests
  • → Real-time activity feed from friends
  • → Per-game discussion forums, live updates
  • → Favourites, recently viewed, avatar picker
  • → Fully responsive — works on iPhone SE

Stack

Flutter Web Frontend
Firebase Auth Authentication
Cloud Firestore Database
Cloud Functions Node.js backend
RAWG API Game data
Steam Web API Library + achievements
Firebase Hosting Deployment

Getting Started

You will need Flutter SDK (3.x+), Firebase CLI and Node.js 18+.

  1. Clone the repo
    git clone https://github.com/Kszlufik/FinalProject.git
cd FinalProject
  2. Install Flutter dependencies
    flutter pub get
  3. Connect Firebase
    Create a project at console.firebase.google.com then run:
    flutterfire configure
  4. Add your RAWG API key
    Get a free key at rawg.io/apidocs and replace the key in lib/services/rawg_service.dart.
  5. Deploy Cloud Functions with your Steam API key
    Steam's API has no CORS headers so all Steam calls go through Firebase Cloud Functions. The API key is stored as a Firebase Secret.
    cd functions
npm install
firebase functions:secrets:set STEAM_API_KEY
firebase deploy --only functions
  6. Create Firestore composite indexes
    Go to Firebase Console → Firestore → Indexes and create two indexes manually. Without these the activity feed and forums return nothing silently.
    # Index 1 — activity feed
Collection: activity
Fields: uid (ASC), timestamp (DESC)

# Index 2 — forum top comment
Collection: forums/{gameId}/posts
Fields: likeCount (DESC), timestamp (DESC)
  7. Run locally
    flutter run -d chrome --web-renderer html
    Note: The --web-renderer html flag is required. CanvasKit causes a black screen on Safari mobile.

Deployment

flutter build web --web-renderer html
firebase deploy --only hosting
Remember: Firestore composite indexes must be created before deploying. They are not created automatically.

Known Issues

  • ! Safari on older iOS versions may have rendering issues with the HTML renderer
  • ! Activity feed queries run sequentially for 10+ friends — Future.wait parallelism planned
  • ! Steam achievements require the user's profile and game library to be set to public on Steam
