Project
PlayPal
A Social Game Discovery and Tracking Platform

PlayPal is a Flutter web application that allows gamers to discover new games, track their playing status, write personal reviews and connect with friends. It integrates the RAWG API for game discovery, Firebase for authentication and data persistence, and the Steam Web API via Firebase Cloud Functions to sync a user's game library and achievements. The platform features a full social layer including friend requests, friend profiles and a personalised user profile with editable details.
