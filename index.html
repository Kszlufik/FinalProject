PlayPal

A Social Game Discovery and Tracking Platform

PlayPal is a web application that lets gamers discover new games, track what they are playing, write personal reviews and connect with friends. Built with Flutter and Firebase, it integrates the RAWG API for game discovery, Steam API for library sync, and Cloud Firestore for real-time data persistence across a full social networking layer.
