# PlayPal

A social game discovery and tracking platform built with Flutter and Firebase.

## About

PlayPal is a web application that lets gamers discover new games, track what they are playing, write personal reviews and connect with friends. The idea came from wanting one place to manage my gaming library and see what friends are playing without jumping between different apps.

## Features

- Browse and search games powered by the RAWG API
- Filter by genre, platform, release date and rating
- Save favourite games to your personal library
- Write reviews with a star rating and status (Playing, Completed, Dropped)
- Connect your Steam account to view your full library and achievements
- Add friends by username and view their profiles
- See your friends reviews and favourite games
- Personal profile page with editable username, bio and avatar
- Secure authentication with email and password via Firebase

## Tech Stack

- Flutter (Web)
- Firebase Authentication
- Cloud Firestore
- Firebase Cloud Functions (Node.js)
- RAWG Video Games API
- Steam Web API

## Getting Started

### Prerequisites

- Flutter SDK installed
- A Firebase project set up with Authentication and Firestore enabled
- RAWG API key

### Running the app
```
flutter run -d chrome
```
